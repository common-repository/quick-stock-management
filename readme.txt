Quick Woocommerce Stock Management using Bulk Action.

== Description ==
Its most quick stock management system, Quick Stock Management allows you manage stock for products using bulk action in product list section. In Quick Stock Management no need to setup any configuration or shortcode.


= Features Included =
* Single Click Configuration.
* No Need to Configuration.
* No Need programming skill to intrigartion.

== Installation ==

*  From WordPress Dashboard go to "Plugins >> Add New >> Uploads", select 'quick-stock-management.zip' file and upload it. Or,
*  Extract the zip file, copy and paste to the 'wp-conten/plugins/' directory of your WordPress Installation.
*  Activate the plugin.
